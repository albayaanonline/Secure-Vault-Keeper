## Rule
The Orval-generated `requestUploadUrl` from `@workspace/api-client-react` takes flat params — NOT wrapped in `{ data: {...} }`:
```typescript
const res = await requestUploadUrl({ name, size, contentType });
// Response field is `uploadURL` (capital URL):
return { method: "PUT", url: res.uploadURL, ... };
```

## Why
React Query mutation hooks use `{ data: {...} }` shape. But the plain fetch functions generated by Orval take the body directly as the argument. `requestUploadUrl` is a plain function, not a mutation hook.

## How to apply
Use `requestUploadUrl({ name, size, contentType })` inside `onGetUploadParameters`. Use mutation hooks (`useCreateVaultFile().mutate({ data: {...} })`) for writes that need cache invalidation.
